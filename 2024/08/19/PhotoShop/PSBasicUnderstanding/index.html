PS 初窥门径

初步认知

基本认识

  1. 打开ps - 主页 - 新建
  2. 整个ps分区域显示
    1. 最顶层-菜单栏
    2. 左侧-工具栏
    3. 顶部第二层-属性栏（父级工具栏）
    4. 右侧-浮动面板（可以长按拖动 随意改变位置）
      1. 打开：菜单 - 窗口 - xx
      2. 重置：菜单 - 窗口 - 工作区 - 复位基本功能
  3. 画布分层
    1. 顶部 - 标题栏
    2. 中间 - 画布
    3. 左下 - 状态栏 更改% 像素
  4. 预设（快捷键）
    1. ctl+ K （mac为 command + k）打开首选项
    2. 首选项 - 文件处理 - 自动存储恢复信息的间隔 - 5分钟
    3. 界面 - 颜色方案 - 黑色

基础操作（图片基本操作）

  1. 置入图片

    1. 单独处理： 文件 - 打开 - 选择本地图片
    2. 添加至已创建文件：
      1. 拖动图片至操作区域
      2. 文件 - 置入嵌入对象 （嵌入源文件）
      3. 文件 - 置入链接的智能对象 （只插入源文件的链接地址，省内存，但发送别人依据地址就会丢失）
      4. 图片进入后会显示一个叉号 说明是一个智能对象文件，可放大缩小，按回车即可确认最终尺寸，叉号消失

  2. 删除 - 选中点击 delete键

  3. 撤回

    1. 右上方 有个回滚箭头标识 点击 展开历史记录 点击对应步骤删除即可
    2. 菜单栏 - 编辑 - 还原删除状态 （快捷键ctl+z）

  4. 放大和缩小

    1. 放大/缩小 option + 上/下滑
    2. ctl + 加号/减号

  5. 图层

    1. 添加图层
      1. 点击右下角 + 号标识
    2. 改变位置
      1. 单个：选中任意单个图层 + 工具栏选中 - 移动工具 + ctl + t 鼠标拖动即可
      2. 整体：shift选中所有图层拖动 + ctl + t 鼠标拖动即可
    3. 删除图层
      1. 拖动至右下角垃圾桶标识
      2. 选中 - delete键

  6. 存储/导出

    1. 文件 - 存储
    2. 文件 - 存储为
    3. 文件 - 导出
    4. 可点击图层处背景前眼睛 关闭背景 导出为透明图

图层认知

图层

  1. 图层过滤器

    1. 图层下第一列 搜索 默认为类型
      1. 类型 右边展开5个选项
        1. 分别为 像素 调整 文字 形状 智能对象
        2. 视为筛选条件 反选即取消 可多选

  2. 不透明度和填充

    1. 图层下第二列 默认为正常 不透明度100%
      1. 设置对应图层的不透明度
    2. 填充 不改变边框，只生效内容

  3. 图层混合模式

    1. 位置：位于 不透明度 左侧选择框
    2. 例如正片叠底 变亮 滤色可将带有背景色图片融入至区域图片中
    3. 作用：提取单个图层一个像素融入其他像素中
    4. 锁定按钮：位于填充左侧
      1. 锁定透明像素
      2. 禁用画笔
      3. 锁定位置
      4. 锁定画板，不能拖拽至其余画板
      5. 锁定全部（以上四种都生效）

  4. 观察图层面板

    1. 实用： 左侧切换移动工具选项，双击文字，右上方可改变颜色
    2. 命名：双击图层
    3. 合并图层
      1. 合并
        1. shift选中合并多个图层 + ctl + e
        2. 注意：会丢失原图层
      2. 盖印
        1. shift选中多个 command+option+e
        2. 不会丢失原图层，出现一个合并图层，内容相当于复制一份在原位置

  5. 图层指令按钮

    1. 链接图层
    2. 添加图层样式
    3. 添加图层蒙层
    4. 创建新的填充或者调整图层
    5. 创建新组
      1. 相当于创建分类 将多个图层放入
    6. 创建图层
    7. 删除图层

工具栏

移动工具

  1. 左侧工具栏第一个选项下第一个选项
    1. 属性栏
      1. 都不勾选状态下（默认选中- 自动选择 ）
        1. 点击对应图层 拖动鼠标 即可改变图层位置
      2. 勾选自动选择 选项默认为图层 切换组即可实现改组下全体图层拖动
      3. 显示变换控件 command + t 显示图层占用大小边框
      4. 右侧对齐方式 需多选图层

画板工具

  1. 左侧工具栏第一个选项下第二个选项
