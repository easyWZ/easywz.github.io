macOS安装

  1. 采用Homebrew安装方式：brew install nginx
  2. 启动Nginx服务：brew services start nginx
  3. 通过浏览器访问http://localhost，如果看到Nginx的欢迎页面，说明安装成功。
  4. 重启Nginx服务brew services restart nginx
  5. 配置文件通常位于/etc/nginx/nginx.conf
