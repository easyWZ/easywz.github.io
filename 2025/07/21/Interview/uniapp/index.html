1. UniApp 是什么？它的运行机制是怎样的？

UniApp 是基于 Vue.js 的跨平台框架，一套代码可以编译到多个平台（如微信小程序、App、H5等）。
它运行机制分为两类：
H5
直接运行在浏览器中，使用 Vue + Web 标准
小程序
编译为各类小程序语法（如微信 wxml + wxss + js）
App（vue）
运行在 WebView 中，前端用 Vue，底层调用原生能力
App（nvue）
使用 Weex 渲染引擎，组件和样式原生渲染，性能更高

3. UniApp 的组件通信方式有哪些？

•	props / emit（父子）
•	$refs（直接操作子组件）
•	Vuex / Pinia（状态管理）
•	eventChannel（页面跳转传参）
•	uni.$on / uni.$emit（全局事件总线）
•	provide / inject（依赖注入）

5. 如何实现一个全局登录弹窗？

•	在 App.vue 中定义 <login-popup v-if="!isLogin" />
•	使用 Vuex/Pinia 管理登录状态
•	拦截未登录操作，统一调出登录弹窗组件

6. 怎么调用原生功能？

小程序平台：使用各平台 API，如 wx.login、tt.getLocation 等
• App 平台：
• 使用 plus API（如 plus.camera、plus.runtime.openURL()）
• 也可封装原生插件（通过 uni_modules）

7. 如何优化长列表性能？

•	使用虚拟列表方案（如 nvue 中的 recycle-list）
•	分批加载（分页）
•	图片懒加载（lazy-load）
•	小程序中避免 setData 频繁更新

8. App 中 H5 性能不佳怎么办？

•	改用 nvue 页面提升性能（原生渲染）
•	减少 dom 层级和图片体积
•	拆分页面避免单页过大
•	尽量使用 v-show 替代 v-if

9. 小程序白屏或卡顿的原因？

•	页面过大（超出组件树深度）
•	动态渲染太多数据 → 使用分段渲染
•	图片太多 / 未压缩
•	setData 次数太多
•	使用第三方 UI 框架未按需引入

10. 如何打包 App？

方法一：使用 HBuilderX：
• 选择【发行】 → 【原生App云打包】
• 填写包名、应用名、证书、权限等
• 自动生成 Android 或 iOS 安装包（apk、ipa）
方法二：本地打包：
• 使用 CLI → npm run build:app-plus
• 使用 Android Studio 或 Xcode 打包

13. UniApp 跨平台原理简述

•	编译阶段根据平台进行模板转换
•	对样式、标签进行平台适配
•	JS 逻辑部分运行在 JS 引擎内（如 App 用 JSCore、浏览器用 V8）
•	不同平台运行时自动接入对应 API（小程序转为 wx.xxx，App 转为 plus.xxx）

14. uniapp 生命周期

  1. 页面生命周期（页面级）:
    onLoad(options)
    页面加载时触发，参数为上个页面传递的参数
    onShow()
    页面显示时触发（进入页面或切回页面）
    onReady()
    页面初次渲染完成后触发
    onHide()
    页面隐藏时触发（如跳转到其他页面）
    onUnload()
    页面卸载时触发（如返回上一级页面）
  2. App 生命周期（仅 App.vue）
    用于监听整个 App 的生命周期，只在 App.vue 中有效：
    onLaunch(options)
    初始化完成时触发，全局只触发一次
    onShow(options)
    应用启动或从后台进入前台时触发
    onHide()
    应用从前台进入后台时触发
  3. 组件生命周期（<script> 中）
    beforeCreate()
    组件实例刚创建
    created()
    组件实例已创建
    beforeMount()
    DOM 挂载前
    mounted()
    DOM 挂载完成
    beforeUpdate()
    数据更新前
    updated()
    数据更新后
    beforeDestroy()
    组件销毁前（已废弃）
    destroyed()
    组件销毁后（已废弃）
    Vue3 中建议使用 onBeforeUnmount 和 onUnmounted 替代上面两个。
  4. 特殊生命周期（小程序 / App 特有）
    onPullDownRefresh()
    下拉刷新回调
    onReachBottom()
    页面触底事件
    onPageScroll(obj)
    页面滚动触发
    onShareAppMessage()
    用户点击右上角分享时触发（小程序）
    onTabItemTap(obj)
    点击 tab 项触发

15. UniApp 和 Vue 的关系？

答：UniApp 是基于 Vue 2（或通过插件支持 Vue 3）的跨平台框架，使用 Vue 语法书写逻辑，但编译时通过 uni compiler 转为各平台代码。

17. 怎么进行 App 的离线打包？

•	使用 HBuilderX → 发行 → 原生App-云打包。
•	或者使用 cli + 自定义基座进行离线打包（需使用 SDK 编译）。

18. 怎么发布到 App Store 或 Android 商店？

1.	使用云打包生成 .apk / .ipa 文件；
2.	Android 提交至：华为、小米、应用宝等；
3.	iOS 需有开发者账号，上传至 App Store Connect 并通过审核。
