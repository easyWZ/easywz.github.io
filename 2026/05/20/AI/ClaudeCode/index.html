Claude Code 安装指南：AI编程助手配置全攻略

本文最后更新于 2026年5月20日 下午

一、安装 Claude Code

1. mac系统安装

  1. macOS / Linux 安装流程（在启动台搜索“终端”点击运行）

  2. 输入命令brew -v查看你的机器是否已安装Homebrew（macOS软件包管理器）

  3. 如果没有安装 Homebrew

    1. 输入以下命令
      /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
    2. 脚本运行后，会自动检测你的系统，并显示一个菜单，让你选择下载镜像源,菜单中通常会显示多个选项；选择 中科大源 或者 清华大学源 ，回车确认
    3. 根据脚本的提示，输入你的 macOS 系统用户密码。输入时屏幕不会显示任何字符，这是正常的安全机制，输完后直接按回车键确认即可
    4. 等待脚本自动完成安装和配置。整个过程通常只需几分钟

  4. 安装好 Homebrew 后，接着输入命令 brew install --cask claude-code

  5. 等待安装完成后，可以输入命令验证 claude --version 显示版本号则表示安装成功.

  6. 启动：直接在终端中输入claude 会让你确认是否授权当前文件夹

  7. 按引导配置，配置完成后输入 /login，浏览器将弹出授权窗口，登录你的 Claude 账号即可完成认证。（没有账号往下看 有免费版使用）

2. windows安装

  1. 打开PowerShell：右键点击“开始”菜单，选择 “Windows PowerShell”（普通模式即可，不一定非要管理员模式），即 windows 终端。
  2. 如果是国内环境，运行命令 winget install Anthropic.ClaudeCode
    1. WinGet是Windows 10/11自带的包管理器
    2. 如果系统提示找不到命令 winget，则需要先前往微软商店（Microsoft Store），搜索并安装 “应用安装程序”
  3. 如果你能翻墙，直接运行命令 irm https://claude.ai/install.ps1 | iex
  4. 验证安装：安装成功后，关闭当前的PowerShell窗口，然后重新打开一个新的PowerShell窗口（无需管理员权限）。看上述mac安装第5步，后续是一样的。

二、安装 ccswitch

点击前往下载安装： https://www.ccswitch.io/zh/
软件作用：统一管理你的 AI 编程工具工作流

三、弄免费账号

  1. 点击前往 https://api.code-relay.com/console/token 没注册就微信扫码注册下
  2. 点击添加令牌
  3. 输入名称，按你的余额设置额度，其他的默认就可以
  4. 复制你的令牌

四、启动claude

配置环境变量：

  1. mac系统打开终端输入(你的API_KEY替换成上面的令牌)

  1. windows复杂些

    设置Windows系统环境变量(在控制面板中)
    需要设置以下三个环境变量，请先获取您的API密钥（即上面的令牌）

    需要设置的环境变量：
    Claude Code配置：
    变量名：ANTHROPIC_BASE_URL
    变量值：https://api.code-relay.com/
    变量名：ANTHROPIC_API_KEY
    变量值：你的密钥
    变量名：ANTHROPIC_AUTH_TOKEN
    变量值：你的密钥
    OpenAI Codex配置（可选）：
    变量名：OPENAI_BASE_URL
    变量值：https://api.code-relay.com/v1
    变量名：OPENAI_API_KEY
    变量值：你的密钥
    设置方法：

    1. 右键”此电脑” → “属性” → “高级系统设置” → “环境变量”
    2. 在”系统变量”区域点击”新建”
    3. 分别添加上述三个变量
    4. 点击”确定”保存所有更改

配置ccswitch

  1. 添加新供应商

  2. 选择自定义配置

  3. 下拉输入信息
    供应商名称：Code Router（可自定义）
    官网链接：https://api.code-relay.com
    API Key：从控制台获取的密钥
    请求地址：https://api.code-relay.com

  4. 点击添加后 再点击启用

  5. 进入终端（不论是mac还是windows）输入 claude 启动开始你的提问吧
