macOS 终端操作：高频快捷键及命令速查

本文最后更新于 2025年9月4日 上午

1、修改文件权限

sudo chmod -R 777 文件名称

2、修改hosts

  1. 终端打开hosts文件命令：
    sudo vim /private/etc/hosts
    sudo vi /etc/hosts
    1. sudo vi /etc/hosts
    2. 提示输入个人电脑密码
    3. 此时输入E，点击enter
    4. 看到hosts，此时输入i，看到命令行下面显示insert，输入自己需要修改的host
    5. 点击esc退出编辑
    6. 按住shift 和 :，输入wq保存退出
  2. 访达中修改hosts
    1. 访达中打开通过command+shift+g找到/etc/hosts
    2. 右键选择打开方式，选择文本编辑/其他编辑器

3、显示“隐藏文件”

  1. 局部：
    操作快捷键 command+shift+.
  2. 全局：
    defaults write com.apple.finder AppleShowAllFiles -bool true
    Mac关闭“显示隐藏文件”命令：
    defaults write com.apple.finder AppleShowAllFiles -bool false

4、查看8080端口占用情况

sudo lsof -i tcp:8080

5、80端口转向8080

（以下34配置在mac重启后需重新加载）

  1. sudo vim /etc/pf.conf
  2. 在rdr-anchor “com.apple/* 后面加上如下这一行（代表将发到80端口的数据转发到8080上,8080可以更改成自己的端口）
    rdr on lo0 inet proto tcp from any to 127.0.0.1 port 80 -> 127.0.0.1 port 8080
  3. 重新加载配置
    sudo pfctl -f /etc/pf.conf
  4. 启动
    sudo pfctl -e

6、查看运行状态

ps -ef | grep nginx 查看nginx运行状态
ps -ef|grep java 查看java占用

7、终止进程

kill -9 pid 例如：kill -9 1234

  1. 查看进程号：ps -ef|grep java
  2. 杀死进程：kill -9 pid

8、访达中访问具体目录

command+shift+g 输入路径即可访问

9、打开强制退出程序窗口

Command+Option+Esc

10、终端基础命令

  1. pwd显示当前所在目录
  2. ls 文件内部列表展示
  3. cd 进入内部某个对象 可按tab健自动补全
  4. Control + U 删除光标前所有内容
  5. Control + K 删除光标后所有内容
  6. Command + K 清空终端显示 不会影响当前输入
  7. Control + Y 粘贴最后一次删除的内容/撤销删除
  8. Control + A 光标移动到行首
  9. cd .. 返回上一级目录
  10. cd ~ 回到用户目录
  11. cd - 回到上一个目录
  12. touch 文件名 创建文件
  13. rm 文件名 删除文件
  14. rm -rf 目录名 删除文件夹
  15. mkdir 目录名 新建文件夹
  16. mv 原文件名 新文件名 重命名
  17. cp 原文件名 新文件名 复制文件
  18. cat 文件名 查看文件内容
  19. open 文件名 打开文件
  20. open . 打开当前目录
  21. open -a 文件名 打开指定文件
